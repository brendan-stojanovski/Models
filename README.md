# Models

Collection of finance and data analytics projects focused on valuation, modeling, and investment analysis.

## Featured Projects

### McDonald's Corporation (MCD) — 5-Year DCF Valuation Model

Built a 5-year discounted cash flow valuation model for McDonald's to estimate intrinsic enterprise and equity value under multiple operating scenarios.

#### Features
- Revenue and operating forecasts
- Discounted cash flow valuation
- Comparable company analysis
- Football field valuation chart
- Sensitivity analysis
- Equity value and implied share price analysis
